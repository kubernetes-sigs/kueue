{{< /blocks/cover >}}
{{% blocks/lead color="primary"%}}

Kueue 是一个云原生的作业队列系统，适用于 Kubernetes 集群中的批处理、HPC、AI/ML 和类似应用。

{{% /blocks/lead %}} {{% blocks/lead color="secondary" %}}

使用 Kueue 构建具有配额和层次结构的多租户批处理服务，在组织内的团队间共享资源。基于可用配额，Kueue 决定作业何时等待以及何时何地运行。

{{% /blocks/lead %}} {{% blocks/lead color="dark" %}}

Kueue 与标准的 kube-scheduler、cluster-autoscaler 和 Kubernetes 生态系统的其余部分结合使用。这种组合允许 Kueue 在本地和云端运行，资源可以是异构的、可替换的和动态配置的。
