### Usage:

```
kubectl kueue resume [TYPE]
```

### Examples:

```bash
# Resume the workload 
kubectl kueue resume workload my-workload
# Resume the ClusterQueue 
kubectl kueue resume clusterqueue my-clusterqueue
```
